# Tests for functions. 

Create environments to run end2end test

```bash
python -m venv venv
```
Actiavation 
```bash
source venv/bin/activate
```

Install packages 
```bash
pip install -r /path/to/requirements.txt
```
Running python

```
python {filename}/end2end.py
```

Retun test fail if any of the test failed.
